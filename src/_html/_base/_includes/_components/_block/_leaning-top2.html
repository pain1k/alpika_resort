ГОРНОЛЫЖНЫЙ
КЛУБ МОРОЗКО

Зимой для маленьких гостей курорта работают склоны детского горнолыжного клуба Морозко, на которых проходят индивидуальные и групповые занятия. Программа тренировок расписана с использованием схем и картинок.
Юные лыжники могут самостоятельно прочитать свой учебный план. Также для запоминания используют стихи, игровые сценарии, яркие вспомогательные игрушки. С ними горнолыжные уроки становятся вдвойне интересными.
